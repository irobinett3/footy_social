Soccer Social Network App

View Development Plan

Overview

Our app is a social media for soccer/football fans. It will have upcoming fixtures, chat rooms, comment sections, trivia games, and fan rooms for fans to use in anticipation/during a game.

The apps like this that exist are either not soccer-specific, require pay, or only provide one of scores/news and trivia. This will combine those functionalities into one free, simple-to-use app.

Data Sources

Most of our data will come from Kaggle/FootballData.uk. There are several different applications with game data and results that will be used to inform our models. For live data (after each game), we will pull from API-Football and other similar APIs to update our tables after games occur.

Basic Functions

  • Insert:
    • Insert Users/Favorite Teams/etc. into db
    • Insert chat message/poll votes
    • Insert game results
    • Store user’s daily game performance
  • Update:
    • Update content of games each day
    • Update scores in live time from APIs
  • Delete:
    • Delete Chat Messages after a week (or other relevant time)
  • Searching DB:
    • Find soonest upcoming games
    • Find most relevant recommendations for user
    • Find relevant data for each game

Advanced Functions

  • @Footyai requests:

    Users in chat can ask for specific stats/predictions/user sentiment. This will require NLP tasks to extract the information and then parsing our database to return values.

  • Game generator:

    Instead of a moderator generating the categories for the daily games, it will be automated. For example: “Top 10 (x), game.” The model can predict “Players with the most points won while playing for Leeds United.”
